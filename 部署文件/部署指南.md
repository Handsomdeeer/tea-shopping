# 潮茶商城生产环境部署指南

## 一、准备服务器

1. 购买云服务器（推荐阿里云/腾讯云/华为云）
   - 配置：2核4G 起步
   - 系统：CentOS 7/8 或 Ubuntu 20.04/22.04
   - 带宽：3Mbps 起步

2. 购买域名（可选，但推荐）
   - 在阿里云/腾讯云/GoDaddy 购买
   - 完成备案（国内服务器必须备案）

3. 配置安全组/防火墙
   - 开放 22（SSH）
   - 开放 80（HTTP）
   - 开放 443（HTTPS）

---

## 二、安装运行环境

### 1. 安装 JDK 17

```bash
# CentOS
yum install -y java-17-openjdk java-17-openjdk-devel

# Ubuntu
apt update
apt install -y openjdk-17-jdk

# 验证
java -version
```

### 2. 安装 MySQL 8.0

```bash
# CentOS
wget https://dev.mysql.com/get/mysql80-community-release-el7-11.noarch.rpm
rpm -Uvh mysql80-community-release-el7-11.noarch.rpm
yum install -y mysql-community-server
systemctl start mysqld
systemctl enable mysqld

# Ubuntu
apt install -y mysql-server
systemctl start mysql
systemctl enable mysql
```

修改 MySQL root 密码：
```bash
mysql_secure_installation
```

创建数据库：
```bash
mysql -u root -p
# 然后执行
CREATE DATABASE chaocha_mall CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;
EXIT;
```

### 3. 安装 Nginx

```bash
# CentOS
yum install -y epel-release
yum install -y nginx
systemctl start nginx
systemctl enable nginx

# Ubuntu
apt install -y nginx
systemctl start nginx
systemctl enable nginx
```

## 三、上传项目文件

### 1. 本地打包

```bash
mvn clean package -DskipTests
```

### 2. 上传到服务器

使用 SCP 或 FTP 工具上传：

```bash
scp target/tea-shopping-0.0.1-SNAPSHOT.jar root@你的服务器IP:/opt/tea-shopping/
scp database/schema.sql root@你的服务器IP:/opt/tea-shopping/
```

在服务器上创建目录：
```bash
mkdir -p /opt/tea-shopping
mkdir -p /var/log/tea-shopping
```

### 3. 初始化数据库

```bash
cd /opt/tea-shopping
mysql -u root -p chaocha_mall < schema.sql
```

## 四、配置生产环境

### 1. 修改生产环境配置

编辑 `src/main/resources/application-prod.properties`：

```properties
spring.datasource.password=你的数据库密码
aliyun.oss.access-key-id=你的阿里云AccessKeyId
aliyun.oss.access-key-secret=你的阿里云AccessKeySecret
aliyun.oss.bucket-name=你的Bucket名称
```

### 2. 重新打包（修改配置后）

```bash
mvn clean package -DskipTests
scp target/tea-shopping-0.0.1-SNAPSHOT.jar root@你的服务器IP:/opt/tea-shopping/
```

## 五、配置系统服务

### 1. 创建 systemd 服务

把 `tea-shopping.service` 上传到 `/etc/systemd/system/`：

```bash
cp tea-shopping.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable tea-shopping
```

### 2. 启动/停止/重启

```bash
systemctl start tea-shopping    # 启动
systemctl stop tea-shopping     # 停止
systemctl restart tea-shopping  # 重启
systemctl status tea-shopping   # 查看状态
```

查看日志：
```bash
journalctl -u tea-shopping -f
```

## 六、配置 Nginx 反向代理

### 1. 修改 Nginx 配置

把 `nginx.conf` 中的 `你的域名` 替换为实际域名，然后：

```bash
cp nginx.conf /etc/nginx/conf.d/tea-shopping.conf
nginx -t
systemctl reload nginx
```

### 2. 配置域名解析

在你的域名管理后台，添加 A 记录：
- 主机记录：@（或 www）
- 记录值：你的服务器 IP

等待 DNS 生效（通常 10 分钟~几小时）。

---

## 七、配置 HTTPS（SSL 证书）

### 1. 申请免费证书

使用 Certbot 申请 Let's Encrypt 免费证书：

```bash
# CentOS
yum install -y certbot python3-certbot-nginx

# Ubuntu
apt install -y certbot python3-certbot-nginx

# 申请证书
certbot --nginx -d 你的域名
```

### 2. 自动续期

Certbot 会自动配置定时任务续期证书，无需手动操作。

---

## 八、安全加固

### 1. 修改默认密码

- 修改数据库 root 密码
- 修改管理员后台默认密码（admin/admin123）
- 修改测试用户密码

### 2. 修改 OSS 密钥

在阿里云控制台：
- 创建新的 AccessKey
- 限制该密钥的权限（只给 OSS 读写权限）
- 更新 `application-prod.properties`

### 3. 防火墙配置

```bash
# 关闭 8080 端口外网访问（只允许 Nginx 访问）
firewall-cmd --permanent --remove-port=8080/tcp
firewall-cmd --reload
```
